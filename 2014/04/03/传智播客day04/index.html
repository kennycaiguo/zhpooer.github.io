javascript

prototype

prototype 是一个类的属性，是类的一个属性, 是类的一个实例，也就是说当一个类被声明后，js自动创建， 并且作为类的prototype属性而存在,称作类的原型

在扩展共用属性，用prototype；在应用个性化的属性时，用构造函数

DOM

dom: 文档对象模型，是一套规范

dom 定义了哪些对象，这些对象有哪些方法和属性

标签和标签之间也都是文本 / 拷贝老师 /！！！！！
