xml语言

  • 作用：传输数据 存储数据

xml结构

  • xml1.1 发一次请求，多次响应
  • xml1.0 发一次请求，一次响应

注意： 大小写敏感

dtd xml约束

  • DOCTYPE: 内部声明
  • ELEMENT: 元素声明

类型

  • CDATA: 字符数据
  • ID: id

约束

  • #REQUIRED
  • #FIXED value: 固定
  • #IMPLIED: 不是必须

CDATA

<![CDATA[will be not parsed content]]>

xml css

<?xml-stylesheet type="text/css" href="book.css"?>

实体 entity

<!ENTITY site "www.itcast.cn">

<publish>&my;</publish> <!-- 引用 -->

混合型

<!ELEMENT note (#PCDATA|to|from|header|message)*>

schema

Schema是一个xml文件扩展为xsd，替代dtd
