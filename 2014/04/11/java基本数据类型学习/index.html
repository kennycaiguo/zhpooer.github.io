基本数据类型对象包装类

为了方便操作基本数据类型, 将其封装成了对象, 在对象中定义了属性和行为丰富了数据的操作. 用于描述该对象的类就称为基本数据包装类型.

数据类型 包装类
byte Byte
short Short
int Integer
long Long
float Float
double Double
char Char
boolean Boolean
