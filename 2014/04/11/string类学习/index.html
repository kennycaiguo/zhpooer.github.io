StringBuffer

StringBuffer 是字符串缓冲,用于存储数据的容器

  1. 长度是可变的
  2. 可以存储不同类型的容器
  3. 最终转换成字符串
  4. 可对字符串进行修改, 可添加 append(data), 可插入 insert(pos, data), 删除 delete(pos, pos)

StringBuilder

StringBuilder 和 StringBuffer 功能和用法一模一样, StringBuffer 线程安全, StringBuilder 线程不安全, 单线程推荐使用 StringBuilder
