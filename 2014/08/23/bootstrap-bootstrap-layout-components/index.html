Bootstrap provides a toolkit of flexible components that can be used in designing application interfaces, web features, and more. All of the plugins are available in one separate JavaScript file, or you can use the Bootstrap customizer to pick and choose which plugins you want.

Dropdown Menus

Dropdown menus are toggleable, contextual menus for displaying links in a list format. The dropdowns can be used on a variety of different elements, navs, buttons, and more.

You can have a single dropdown or extend the dropdown into another submenu.

Dropup Menus

Menus can also be built to drop up rather than down . To make this change, simply add .dropup to the .btn-group container. To have the button pull up from the righthand side, add .pull-right to the .dropdown-menu

Breadcrumbs

Breadcrumbs are a great way to show hierarchy-based information for a site. There is a also a helper class of .divider that mutes the colors and makes the text a little smaller.

Pagination

Bootstrap handles pagination like a lot of other interface elements, an unordered list, with wrapper a <div> that has a specific class that identifies the element.

You can add .pagination-centered to the parent <div>. This will center the contents of the <div>. If you want the items right-aligned in the <div>, add .pagination-right.

For sizing, in addition to the normal size, there are three other sizes that can be applied by adding a class to the wrapper <div>: .pagination-large, .pagination-small, and .pagination-mini

Badges

Badges are similar to labels

Typographic Elements

In addition to buttons, labels, forms, tables, and tabs, Bootstrap has a few more elements for basic page layout.

Hero Unit

The hero unit is a large content area that increases the size of headings and adds a lot of margin for landing page content

Page Header

The page header is a nice little feature to add appropriate spacing around the headings on a page.

Thumbnails

To create a thumbnail, add an <a> tag with the class of .thumbnail around an image. This adds four pixels of padding and a gray border

Alerts

To close the alert, you can use a button that contains the data-dismiss="alert" attribute.

They are added by using either .alert-error, .alert-success, or .alert-info.

Progress Bars

In addition to the blue progress bar, there are options for green, yellow, and red using the .bar-success, .bar-warning, and .bar-danger classes.

Media Object

When you look at social sites like Facebook, Twitter, and others, and strip away some of the formatting from timelines, you will see the media object

Miscellaneous

Some of these components are layout-based, and a few are production-based helper classes.

Wells

A well is a container <div> that causes the content to appear sunken on the page.

There are two additional classes that can be used in conjunction with .well: .well-large and .well-small.
